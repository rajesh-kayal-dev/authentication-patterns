# Email Verification Auth - JWT

## Features
- User Registration
- Email Verification via JWT link
- Login System
- Protected Routes
- Logout
- Token-based Authentication

## Tech Stack
- Node.js + Express
- MongoDB
- JWT
- React (Vite + TypeScript)
- Axios

##  Structure
backend/ → API  
frontend/ → React app  

## Setup

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev  

## Flow
1. Register → email sent  
2. Verify email  
3. Login → get token  
4. Access protected routes
